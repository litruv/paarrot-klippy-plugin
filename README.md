# Klipy GIF Search

Search and send GIFs with a clickable picker overlay.

## Usage

### `/gif [search]`
Shows an interactive GIF picker with clickable thumbnails.

**Examples:**
- `/gif dancing cat`
- `/gif celebration`

Click any GIF to send it immediately.

## Settings

- **Klipy API Key** - Get from [partner.klipy.com](https://partner.klipy.com)
- **Results Limit** - Number of GIFs to show (default: 10)
- **GIF Size** - Original/Medium/Small
